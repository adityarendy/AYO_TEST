Cara penggunaan Automation Script
    1. Instal Visual Studio Code https://code.visualstudio.com/download
    2. Pilih sesuai platform yang digunakan
    3. Jalankan Visual Studio Code
    4. Download dan install Python https://www.python.org/downloads/
    5. Cek instalasi Python melalui terminal
    6. ketik py --version pada terminal, pastikan versi python sesuai lalu enter
    7. ketik git clone https://github.com/adityarendy/AYO_TEST.git pada terminal visual code lalu enter
    8. Ketik git status pada terminal lalu enter, pastikan branch pada Main
    9. Click pada File ayo_test.py
    10. Click Run pada bagian pojok kanan atas untuk running script
